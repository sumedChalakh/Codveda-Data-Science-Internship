##  Projects Included

###  1. Sentiment Analysis (Natural Language Processing)

**Objective:**  
Classify text data into sentiment categories by applying NLP preprocessing techniques and supervised machine learning models.

**Key Steps:**
- Text cleaning and normalization
- Tokenization and stopword removal
- Feature extraction using TF-IDF
- Model training using:
  - Naive Bayes
  - Logistic Regression
- Model evaluation using Precision, Recall, and F1-score

**Skills Demonstrated:**
- Handling unstructured text data
- Feature engineering for NLP
- Evaluation of classification models under potential class imbalance

**Notebook:**  
`sentiment_analysis.ipynb`

---

### 2. Time Series Analysis & Forecasting

**Objective:**  
Analyze time-dependent data and forecast future values using statistical time series models.

**Key Steps:**
- Time series visualization
- Trend, seasonality, and residual decomposition
- Baseline forecasting using moving average
- Forecasting using ARIMA model
- Performance evaluation using RMSE

**Skills Demonstrated:**
- Working with time-indexed data
- Identifying patterns in sequential data
- Building and evaluating forecasting models

**Notebook:**  
`time_series_forecasting.ipynb`
