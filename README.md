# Codveda Data Science Internship Projects

##  Internship Overview
This repository contains the projects completed during my **Data Science Internship at Codveda Technologies**.  
The internship focused on building a strong foundation in data science by applying concepts through **hands-on, structured tasks** covering data preprocessing, exploratory data analysis, machine learning, natural language processing, and time series forecasting.

The work emphasizes **end-to-end problem-solving**, proper evaluation, and clear communication of insights.

##  Tools & Technologies
- **Programming Language:** Python  
- **Data Analysis:** Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn

- **Natural Language Processing:** TF-IDF, Naive Bayes, Logistic Regression  
- **Time Series Analysis:** ARIMA (Statsmodels)

##  Level 1: Data Preprocessing & Exploratory Data Analysis

### Data Cleaning & Preprocessing
- Inspected dataset structure and data types
- Checked for missing values and duplicates
- Detected outliers using the IQR method
- Encoded categorical variables
- Standardized numerical features  

**Dataset:** Iris Dataset

### Exploratory Data Analysis (EDA)
- Statistical summaries
- Feature distribution analysis
- Species-wise comparisons
- Correlation analysis
- Key insights for feature importance

---

##  Level 2: Supervised Machine Learning

### House Price Prediction (Regression)
- Built baseline and advanced regression models
- Models used: Linear Regression, Random Forest
- Evaluated using RMSE and RÂ² score
- Analyzed feature importance to identify key price drivers  

**Dataset:** House Price Prediction Dataset

### Iris Species Classification
- Developed classification models
- Models used: Logistic Regression, Random Forest
- Evaluated using Accuracy, Precision, Recall, F1-score, ROC-AUC
- Used confusion matrix for error analysis  

**Dataset:** Iris Dataset

---

##  Level 3: Advanced Data Science

### Sentiment Analysis (NLP)
- Cleaned and preprocessed text data
- Converted text into numerical features using TF-IDF
- Models used: Naive Bayes, Logistic Regression
- Evaluated using Precision, Recall, and F1-score  

**Dataset:** Sentiment Dataset

### Time Series Forecasting
- Created and analyzed time-indexed data
- Visualized trends and patterns
- Applied moving average as a baseline
- Built ARIMA model for forecasting
- Evaluated forecasts using RMSE
